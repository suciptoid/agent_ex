- **B1: Enter to send**: Added `.ChatInput` colocated JS hook — Enter submits the form; Shift+Enter inserts a newline.
- **B2: Discord/Slack-style messages**: Rewrote message list from card layout to compact list items with avatar, sender name, timestamp, and content per row.
- **B3: Tool call truncation**: Added `.ToolCallMessage` colocated JS hook that truncates long tool messages to 5rem with a "Show more / Show less" toggle.

### New Features
- **C2: LLM logging**: Added `Logger.debug/info/error` calls throughout `Runner`, `Orchestrator`, and `Tools`.
- **C3: Streaming LLM responses**: Added `run_streaming/4` to `Runner`. Agents without tools use `ReqLLM.stream_text` for real token-by-token streaming; agents with tools use the tool loop then emit the complete response. `show.ex` uses `Task.async` + `handle_info` to stream tokens into `@streaming_content` with a blinking cursor in the template.
- **C4: Multi-agent handover**: `Orchestrator` injects an agent roster into the system prompt and creates a dynamic `handover` tool (closure with agent map + message snapshot) for rooms with >1 agent. `Runner` accepts `:extra_tools` and `:extra_system_prompt` opts.

### Tests
